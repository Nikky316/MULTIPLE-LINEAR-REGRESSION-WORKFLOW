# Multiple Linear Regression – Marketing Analysis

## Dataset
Contains marketing spend across TV, Radio, Social Media, and Influencer category with Sales as target.

## Goal
Predict Sales and optimize marketing spend allocation.

## Tools Used
- pandas
- seaborn
- statsmodels
- scipy

## Setup
pip install pandas seaborn statsmodels scipy matplotlib

## Key Findings
- Significant predictors identified using p-values
- Multicollinearity checked using VIF
- Model validated via residual diagnostics

## Recommendation
Prioritize channels with highest statistical impact on Sales.
